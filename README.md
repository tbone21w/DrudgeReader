# DrudgeReader

Fun little app based on API friend built and first somewhat useful thing I built on IOS.

* Swift
* CoreData
* NSFetchedResultsController with Search Predicates and sorts 
* Local Notifications
* Settings
* Custom Graphics
* Cocoapods
